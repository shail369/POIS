## PA#1b — OWF from PRG (Proper Reduction)

### Claim
Let G : {0,1}^n -> {0,1}^{n + l} be a secure PRG.  
Define f(s) = G(s). Then f is a one-way function.

---

### Proof (by reduction)

Assume, for contradiction, that f is **not** one-way.  
Then there exists a PPT adversary A such that:

Given y = G(s), A outputs s' with G(s') = y  
with non-negligible probability.

---

### Reduction: Build distinguisher D using A

We construct a PPT distinguisher D that breaks PRG security.

D is given input y, where:
- either y = G(s) for random s (PRG case), or
- y is uniform random in {0,1}^{n+l}

D works as follows:

1. Run A(y) to obtain s'
2. If G(s') == y:
       output "PRG"
   else:
       output "Random"

---

### Analysis

Case 1: y = G(s) (PRG output)

- By assumption, A finds s' such that G(s') = y
  with non-negligible probability
- So D outputs "PRG" with non-negligible probability

---

Case 2: y is uniform random

- With overwhelming probability, y is not in the image of G
- So for any s', G(s') != y
- Hence D outputs "Random" with overwhelming probability

---

### Conclusion

D distinguishes PRG output from uniform random with non-negligible advantage,  
which contradicts the security of G.

Therefore, our assumption was false.

Hence, f(s) = G(s) is a one-way function.
